# Entity: SkillsBuilder Project

## Description
SkillsBuilder is a meta-development workspace designed to build, optimize, and maintain high-quality AI Agent Skills. It serves as the primary factory for skills used in the Antigravity IDE.

## Core Components
*   **Skill Architect**: The engine that generates `SKILL.md` files based on predefined patterns.
*   **LLM Wiki**: A persistent knowledge base (`wiki/`) that stores synthesized project insights.
*   **SOP**: A standard operating procedure for cross-project development.
*   **Global KI**: Registered as a Knowledge Item in the Antigravity system core for universal access.

*   **Architectural Standards**:
    - **MECE File Management**: Mutually Exclusive, Collectively Exhaustive project structure.
    - **PDCA Cycle**: Plan-Do-Check-Act recorded in `DEV_LOG.md`.
    - **GDD (Graph-Driven Dev)**: Structural awareness via GitNexus integration.
    - **Surgical Boundary**: Minimalistic changes with zero side effects.
