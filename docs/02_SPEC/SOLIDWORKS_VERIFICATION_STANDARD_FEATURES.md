## 3. 各板塊詳細驗證標準

### 3.1 歡迎使用 (Welcome)

**SolidWorks 說明對應**: 歡迎使用 SOLIDWORKS 線上說明

| 驗證項目 | 測試方法 | 通過標準 | 等級 |
|----------|----------|----------|------|
| 歡迎對話方塊啟動 | 啟動 3D-Builder | 顯示包含「新增功能」「學習單元」「論壇」「MySolidWorks」的歡迎面板 | L2 |
| 學習單元導航 | 點擊學習單元 | 依序展示從基礎到進階的教學範例 | L2 |
| 互動式新增功能 | 啟用互動式新功能 | 每個新增/變更功能旁顯示資訊圖示，點擊後顯示詳細說明 | L2 |
| 新版本說明 | 點擊「新版本說明」 | 顯示技術要求、已知問題、文件變更列表 | L2 |
| 功能表列存取 | 游標移至主標誌 | 顯示功能表列，包含「檔案」「編輯」「檢視」等菜單 | L2 |
| 快速導覽 (Quick Tour) | Welcome 對話方塊→學習標籤 | 啟用互動式課程，引導使用者完成基本操作 | L2 |

**Golden Part**: 無（UI 流程驗證）

### 3.5 從 2D 到 3D (2D to 3D)

**SolidWorks 說明對應**: 從 2D 到 3D

此板塊是 3D-Builder 的核心功能，需要最嚴格的 L4 驗證等級。

| 驗證項目 | 測試方法 | 通過標準 | 等級 |
|----------|----------|----------|------|
| 草圖平面選擇 | 選擇基準面開始草圖 | 成功進入草圖模式，3D 視口顯示參考基準面 | L3 |
| 草圖工具完整性 | 使用直線、圓、弧、矩形、多段線等 | 每種工具生成的圖元正確無誤 | L3 |
| 幾何約束 | 添加平行、垂直、重合、相切、同心、對稱 | 約束正確應用，視覺狀態正確（黑/藍/紅） | L4 |
| 尺寸約束 | 標註線性、角度、半徑、直徑尺寸 | 尺寸值正確驅動幾何形狀變化 | L4 |
| 矩形工具 | 繪製 3 種模式（二點、中心點、3點） | 矩形邊界與約束正確 | L3 |
| 圓工具 | 繪製 3 種模式（中心-端點、3點、相切-半徑） | 圓心、半徑、位置正確 | L3 |
| 橢圓工具 | 完整/焦點模式 | 橢圓參數正確 | L3 |
| 直線/捕獲 | 畫直線時自動捕捉端點、中點、節點 | 捕捉準確，捕捉指示器顯示正確 | L3 |
| 多段線 | 混合繪製直線段和弧段 | 連接處連續無縫 | L3 |
| 多邊形 | 繪製正多邊形 | 邊數、大小、方向正確 | L3 |
| 槽口 | 繪製槽口（水平/垂直/圓心/角度） | 槽口尺寸與位置正確 | L3 |
| 中心線 | 添加/移除中心線 | 中心線以虛線顯示，不參與實體生成 | L2 |
| 原點顯示 | 顯示/隱藏原點 | 基準原點正確顯示/隱藏 | L2 |
| 動態拖拽預覽 | PBD 求解即時反饋 | 拖拽時 60 FPS 即時視覺反饋 | L3 |
| 精確求解 | 滑鼠釋放後 Newton-Raphson 迭代求解 | 求解殘差 < 10^-6 mm，達到微米級精度 | L4 |

**Golden Part**: 包含所有草圖工具的綜合草圖→Extrude→完整特征樹
