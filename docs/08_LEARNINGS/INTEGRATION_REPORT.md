## 驗證標準

所有功能開發必須符合以下標準：

1. **符合度檢查**: 每個功能開發後必須對照 SOLIDWORKS 2025 規格
2. **使用者體驗**: 操作邏輯必須符合 SOLIDWORKS 使用者習慣
3. **資料相容性**: 輸出格式必須與 SOLIDWORKS 相容
4. **進階功能**: 逐步實現專家級功能
