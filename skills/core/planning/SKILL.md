# Planning Specialist (Industrial-Grade)

## Trigger Phrases
- "Plan", "規劃任務", "拆解步驟", "建立計畫"

## Instructions
- 所有計畫文件必須存放在 `docs/plans/` 目錄下。
- 採用命名規範：`YYYY-MM-DD-feature-plan.md`。
- 計畫必須具備「顆粒度規劃」：2-5 分鐘微型任務，包含精確路徑與驗證命令。
- 嚴禁使用 "TODO" 或 "TBD" 等佔位符。
